# 챙겨 앱 디자인 시안 (3가지)

---

## 시안 1: 미니멀 모던 (Minimal Modern)

### 컨셉
깔끔하고 직관적인 UI로 정보 전달에 집중. 불필요한 요소를 제거하고 화이트 스페이스를 활용한 여유로운 레이아웃.

### 컬러 팔레트
```
주 색상 (Primary):    #4A90E2 (밝은 파란색)
보조 색상 (Secondary): #F5A623 (따뜻한 오렌지)
배경 (Background):    #F8F9FA (밝은 회색)
카드 배경 (Card):      #FFFFFF (흰색)
텍스트 (Text):        #2C3E50 (진한 회색)
텍스트 보조 (Muted):   #95A5A6 (연한 회색)
경고 (Warning):       #E74C3C (빨강)
성공 (Success):       #2ECC71 (초록)
```

### 타이포그래피
- **헤더**: SF Pro Display / Noto Sans KR Bold (24-32px)
- **본문**: SF Pro Text / Noto Sans KR Regular (14-16px)
- **캡션**: SF Pro Text / Noto Sans KR Light (12px)

### 화면 레이아웃

## 시안 2: 날씨 감성 (Weather Mood)

### 컨셉
날씨에 따라 변하는 그라데이션 배경. 감성적이고 몰입감 있는 경험. 날씨가 앱의 분위기를 만든다.

### 컬러 팔레트 (날씨별)

#### 맑음 (Clear)
```
배경 그라데이션: #87CEEB → #FFE4B5 (하늘색 → 복숭아색)
텍스트: #FFFFFF (흰색)
카드 배경: rgba(255, 255, 255, 0.2) (반투명 흰색)
```

#### 흐림 (Cloudy)
```
배경 그라데이션: #757F9A → #D7DDE8 (회색 → 연한 회색)
텍스트: #FFFFFF
카드 배경: rgba(255, 255, 255, 0.15)
```

#### 비 (Rainy)
```
배경 그라데이션: #2C3E50 → #4CA1AF (어두운 파랑 → 청록)
텍스트: #FFFFFF
카드 배경: rgba(255, 255, 255, 0.1)
```

#### 눈 (Snowy)
```
배경 그라데이션: #E0EAFC → #CFDEF3 (밝은 파랑)
텍스트: #2C3E50 (진한 회색)
카드 배경: rgba(255, 255, 255, 0.3)
```

### 화면 레이아웃

#### 1. HomeScreen (홈 화면)

## 시안 3: 다크 모던 (Dark Modern)

텍스트 (Text):        #FFFFFF (흰색)

경고 (Warning):       #FF3B30 (빨강)

## 비교 및 추천

| 특징 | 시안 1 (미니멀) | 시안 2 (감성) | 시안 3 (다크) |
|------|----------------|--------------|--------------|
| **분위기** | 깔끔, 전문적 | 감성적, 몰입감 | 프리미엄, 모던 |
| **타겟 사용자** | 모든 연령대 | 20-30대 감성파 | 20-30대 기술파 |
| **개발 난이도** | ⭐⭐ (쉬움) | ⭐⭐⭐ (보통) | ⭐⭐⭐⭐ (어려움) |
| **유지보수** | 쉬움 | 보통 | 어려움 |
| **차별화** | 낮음 | 높음 | 매우 높음 |
| **배터리** | 보통 | 보통 | 우수 (다크 모드) |
| **접근성** | 우수 | 보통 | 우수 (높은 대비) |

### 🏆 추천: 시안 1 (미니멀 모던)

**이유:**
1. **빠른 개발**: 표준 React Native 컴포넌트 활용
2. **직관성**: 정보 전달에 집중, 사용자 혼란 최소화
3. **확장성**: 나중에 다크 모드 추가 용이
4. **안정성**: 검증된 디자인 패턴 사용

**향후 발전 방향:**
- v1.0: 시안 1로 MVP 출시
- v1.5: 다크 모드 옵션 추가 (시안 3 요소)
- v2.0: 날씨별 테마 추가 (시안 2 요소)
